# Heart Disease UCI - Data Mining Project

## Overview

This repository contains a collection of data mining and exploratory data analysis (EDA) tasks performed on the Heart Disease UCI dataset.

The goal of this project is to practice data analysis, feature exploration, visualization techniques, and data preparation methodologies that are commonly used in machine learning and healthcare analytics.
